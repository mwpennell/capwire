Create a gamma distribution of capture rates for simulation

Usage

drawCapRatesGamma(shape, rate)

Arguments

shape
gamma shape parameter
rate
gamma rate parameter

Create a gamma distribution of capture rates for simulation

Value

a function which takes an argument n that can be then used to simulate capture rates with simCapture

Description

Can be used in conjunction with simCapture

Examples

## Specify a gamma distribution
gd <- drawCapRatesGamma(1,0.5)
simCapture(n=20, s=100, gd)
